# Solaris Nord Energia ☀️

A **Solaris Nord Energia** é uma plataforma digital de alta performance desenvolvida para conectar clientes residenciais e comerciais a soluções sustentáveis de energia solar no Nordeste brasileiro.

Este projeto foi concebido para demonstrar minha capacidade de entregar soluções web modernas, focadas em conversão, performance e experiência do usuário (UX).

## 🚀 O Desafio

O setor de energia solar enfrenta um grande desafio: converter visitantes em leads qualificados. O objetivo deste projeto foi criar uma landing page que combine:

* **Velocidade extrema:** Carregamento otimizado.
* **Identidade Visual Forte:** Design que transmite confiança e modernidade.
* **Foco em Conversão:** Estrutura pensada para maximizar o contato via WhatsApp e captação de dados.

## 🛠 Tecnologias Utilizadas

* **Framework:** Next.js 16 (App Router)
* **Linguagem:** TypeScript
* **Estilização:** Tailwind CSS v4
* **Componentização:** Lucide React
* **Tematização:** next-themes (Suporte Dark/Light)
* **Feedback de Usuário:** Sonner

## ✨ Diferenciais do Projeto

* **Arquitetura Orientada a Performance:** Uso estratégico de CSS customizado e otimização de renderização.
* **Design System Responsivo:** Interfaces que se adaptam perfeitamente do mobile ao desktop.
* **Acessibilidade e UX:** Foco em navegação fluida, estados de interação (hover/active) e tratamento de erros (Página 404 personalizada).
* **Código Limpo:** Estrutura modular de componentes, seguindo as melhores práticas de desenvolvimento com React.

## 📦 Como rodar localmente

1. **Clone o repositório:**
```bash
git clone https://github.com/emersoncarneirodasilva/solaris-nord-energia.git

```


2. **Entre na pasta do projeto:**
```bash
cd solaris-nord-energia

```


3. **Instale as dependências:**
```bash
npm install

```


4. **Inicie o servidor de desenvolvimento:**
```bash
npm run dev

```


5. **Acesse no navegador:**
Abra [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) para visualizar.
